# Agenda de Contatos em PHP

Sistema simples de agenda de contatos com login de usuários, desenvolvido em PHP orientado a objetos. Cada usuário acessa apenas seus próprios contatos.

## Requisitos
PHP 8.0 ou superior  
MySQL ou MariaDB  
Apache (XAMPP, WAMP ou similar)

## Como rodar o projeto
1. Clone o repositório:
git clone https://github.com/cauegz/login-orientado-a-objetos.git

2. Coloque a pasta do projeto dentro do htdocs (ou pasta equivalente).

3. Crie um banco de dados no MySQL com o nome de `loginoo`.

4. Importe o arquivo SQL que está no projeto.

5. Configure a conexão com o banco em config/conexao.php.

6. Inicie o Apache e o MySQL.

## Funcionalidades
Cadastro de usuários  
Login com sessão  
Agenda de contatos por usuário  
CRUD de contatos

## Tecnologias
PHP (POO)  
MySQL  
HTML/CSS
